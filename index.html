Configure Audio Pronunciation Link MP3 files from your Google Drive folder
How it works:
Name your MP3 files using the English Transliteration for each word (e.g. kukka.mp3, dEham.mp3). Upload them to a shared Google Drive folder, then paste the Google Drive Folder ID below.

To get a direct streamable link for Google Drive files, each file must be shared publicly ("Anyone with the link → Viewer"). The system will construct links as: https://drive.google.com/uc?export=download&id={FILE_ID}

Found in your Google Drive folder URL:
drive.google.com/drive/folders/[FOLDER-ID]
Note: Individual file IDs are needed for direct playback. Use the Base URL option below for simpler hosting.

Files should be named {transliteration}.mp3 (e.g. kukka.mp3).
The app will try loading [base-url]{transliteration}.mp3 for each word. This works best with web-hosted files or a static server.

Paste a JSON map of "transliteration": "google-drive-file-id". Get each file's ID from its shareable link: drive.google.com/file/d/[FILE-ID]/view
